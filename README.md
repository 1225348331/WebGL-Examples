## WebGL 编程指南

### 简介

此项目用来记录《WebGL 编程指南》的学习案例，采用了常用的技术栈如：**Vue3、TypeScript、Vue-Router、Ant-Design-Vue、TWGL.js**。

`ps:本项目采用WebGL2技术进行案例实现,有些示例代码并不是很完美,后续会优化~`

### 项目开发

```
# 克隆项目
git clone https://github.com/1225348331/WebGL-Study.git

# 进入项目目录
cd WebGL-Study

# 安装依赖(推荐使用yarn安装)
yarn install

# 启动服务
yarn run dev
```
